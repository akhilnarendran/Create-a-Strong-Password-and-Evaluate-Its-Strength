In today's digital world, passwords are the first line of defense against unauthorized access to personal and professional accounts. This task focuses on understanding what makes a password strong and how to evaluate its effectiveness against potential cyber threats.


***OBJECTIVE :***

The goal of this project is to explore the principles of password security by creating passwords with varying complexity and testing their strength using online tools. Through this exercise, you will gain insights into best practices for password creation and learn about common password attacks, such as brute force and dictionary attacks.


**Key Activities:***

    1 Create Passwords: Generate multiple passwords with different combinations of uppercase letters, lowercase letters, numbers, symbols, and varying lengths.

    2 Evaluate Strength: Use free online tools like passwordmeter.com to assess the strength of each password.

    3 Analyze Results: Note the scores and feedback provided by the tools to understand what contributes to a password's robustness.

    4 Research Attacks: Investigate common password attacks and how password complexity can mitigate these risks.

    5 Summarize Learnings: Compile a report highlighting best practices and tips for creating strong passwords.

    -----------------------------------------------------------
    1. Short & Simple (Weak)
        cat
      Characteristics: Very short, lowercase letters only, common word.
      Vulnerable to: Dictionary attacks, brute force.

    2. Short with Numbers (Still Weak)
        hello123
      Characteristics: Short, lowercase + numbers, predictable pattern.
      Vulnerable to: Basic brute force, dictionary attacks with number suffixes.

    3. Medium Length, Letters Only (Moderate)
        SunshineDay
      Characteristics: Medium length, mixed case, but no numbers/symbols.
      Vulnerable to: Advanced dictionary attacks.

    4. Medium Length with Numbers (Moderate)
        Winter2024!
      Characteristics: Medium length, uppercase start, numbers, one symbol.
      Better, but predictable structure.

    5. Long Passphrase (Strong)
          CorrectHorseBatteryStaple
      Characteristics: Long, multiple words, no symbols/numbers.
      Resistant to brute force but vulnerable if common phrase.

    6. Random Characters (Strong)
          xQ3!9zL2$r
      Characteristics: Short but random, mixed case + symbols + numbers.
      Hard to guess but harder to remember.

    7. Long & Complex (Very Strong)
          J#b7R$pK9!Lm2@W5

      Characteristics: Long, fully random, mixed case + symbols + numbers.
      Highly resistant to attacks.

    8. Common Password (Weak)
          password123!

      Characteristics: Common base word with minor variations.
      Easily cracked by hackers.

    9. Leet-Speak Substitution (Moderate)
          P@ssw0rd!

      Characteristics: Replaces letters with symbols/numbers (leet-speak).
      Better than plain "password" but still predictable.
